git init

Ajouter tous les fichiers :
git add .

git status

git commit -m "Version initiale"

git remote add origin git@github.com:DTYS65/Projet8-OpenClassRooms.git

git branch -M main

git push -u origin main


Commandes groupées :
git add . && git commit -m "Version finale" && git push -u origin main


Si erreur, par exemple si j'ai entretemps rajouté un ReadMe : The remote repo was initialized/updated separately (e.g., README added on GitHub)

Pull the remote changes first

git pull

Attention il apparait un message : Astuce (éviter ce message à l’avenir)
git pull --rebase

If you want a cleaner history (rebase instead of merge)
git pull --rebase

Puis refaire :
git push

If you’re sure you want to overwrite the remote (⚠️ dangerous)
git push --force


Pour annuler les commit
git resert --hard


Si error: You have not concluded your merge (MERGE_HEAD exists).
faire alors un commit
git commit -m "..."

puis faire le pull 
git pull --rebase


Commandes séparées par ; 
la commande suivante s'exécute toujours

Commandes séparées par && 
la commande suivante ne s'exécute qu'en cas de succès de la commande précédente


New branch :
git branch version_1
git checkout version_1
git add test_version_1.txt
git commit -m "New branch version_1"
git push -u origin version_1

voir toutes les branches :
git branch
git checkout main


Fusionner la branche version_3 avec la branche main :
git checkout main
git merge version_3
